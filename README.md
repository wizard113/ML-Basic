# 📘 ML-Basic

머신러닝 및 딥러닝 기초를 학습하기 위한 실습 자료입니다.  
Python 기반으로 NumPy, Pandas, TensorFlow 등을 활용하여 단계적으로 학습합니다.

---

## 📂 프로젝트 구성

### 🔹 기초 머신러닝
- `1_k-NearestNeighbor.ipynb` : KNN 알고리즘
- `2_선형회귀.ipynb` : 선형 회귀 (Linear Regression)
- `3_로지스틱_회귀.ipynb` : 로지스틱 회귀 (Logistic Regression)
- `4_신경망_기초.ipynb` : 신경망 기본 개념

---

### 🔹 딥러닝 (Neural Network)
- `5_wine_nn.ipynb` : 와인 데이터 기반 신경망
- `6_mnist_nn.ipynb` : MNIST 손글씨 분류
- `6-1_fashion_mnist_nn.ipynb` : Fashion MNIST 분류

---

### 🔹 CNN (합성곱 신경망)
- `7_mnist_cnn.ipynb` : MNIST CNN 모델
